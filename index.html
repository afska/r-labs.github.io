Video games

MouseWars.io (2017~2018)

MouseWars.io is a real time multiplayer cursors war game. The cursors have to fight until only one remains.

Features:

  • Web-based and available online for free
  • Two game modes: All vs all and Team vs team
  • Two score types: Deathmatch and Survival
  • 9 types of attacks
  • Up to 8 players per room
  • In-game chat
  • Servers for South America, North America and Europe
  • Local multiplayer through a Desktop version
  • No login needed!
(It's a remake of MouseWars Classic)

Halu (2015~2016)

An arcade-endless-runner game for Android. The only objetive is survive as far as you can, and beat your friend's scores.

Many dangers exists in Halu's world. There are rocks that can hurt any living thing. You can avoid them by jumping or shrinking, but don't shrink too much... you can disappear! Also, your soul will try to escape and you will have to keep it at your center. The speed will be increased all the time, and the smoke balls will be waiting for you. Be prepared.

Tools

quickp2p (2019)

A library which simplifies connecting two web browsers through WebRTC technologies without the need of a signalling server.

dropbox-universal-backup-tool (2015)

A command line directory synchronizer with Dropbox. It syncs the files from disk to the cloud, with the added feature of being able to preview changes before pushing.

Experiments

ElDumper (2019)

A Nintendo GameBoy cartridge dumper that can read the ROM and read/write battery-backed RAM to manage games and save files.

Arduino MIDI Player (2014)

A MIDI player with buzzers, and a proof of concept about controlling an Arduino board with JavaScript. We all love MIDIs <3


Old stuff

I wrote these projects when I was learning programming, so I don't take any responsability for the quality of any of them :)

They are written in Visual Basic 6, compiled and tested on Windows XP.

MouseWars Classic (2009)

An online two-players war game, where cursors have to defeat each other by using the assigned attacks. It works by using TCP/IP, but a Bot is available for solo playing.

HackTheSystem (2004~2007)

It's an... hmm... a "remote administration" tool.

It has many features like screen controlling and sharing, remote command execution, window management, keylogging, a file manager and a little chat.

El Chat (2004~2005)

A chat client that we used a lot with my group of friends, inspired by the old MSN Messenger. It was my first contact with socket programming and I had a lot of fun doing it.
