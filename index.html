Video games

Halu (2015~2016)

An arcade-endless-runner game for Android. The only objetive is survive as far as you can, and beat your friend's scores.

Many dangers exists in Halu's world. There are rocks that can hurt any living thing. You can avoid them by jumping or shrinking, but don't shrink too much... you can disappear! Also, your soul will try to escape and you will have to keep it at your center. The speed will be increased all the time, and the smoke balls will be waiting for you. Be prepared.

dropbox-unix-backup-tool (2015)

A command line directory synchronizer with Dropbox. It syncs the files from disk to the cloud, with the feature of previewing changes before commit (similar to git status's output).

MouseWars Classic (2009)

An online two-players war game, where cursors have to defeat each other by using the assigned attacks. It works by using TCP/IP, but a Bot is available for solo playing.

Admin permissions and compatibility mode must be enabled in order to play in a modern machine.

A new version is being developed!
